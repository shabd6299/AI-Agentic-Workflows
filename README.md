Autonomous AI Agent built with n8n, Groq,gemini LLM, and Gmail API for Telegram-based business automation
# 🤖 Jarvis: Agentic AI Assistant
This project is an autonomous AI Agent built using **n8n** that manages business operations via **Telegram**.

## 🚀 Features
* **Email Intelligence:** Fetches unread emails and summarizes them in a Markdown table.
* **Bidirectional Control:** Send and reply to emails directly from Telegram.
* **AI Vision:** Real-time image analysis using Vision LLMs.

## 🛠️ Tech Stack
* **Automation:** n8n (Self-hosted via Docker)
* **LLMs:** Groq, Gemini
* **APIs:** Gmail API, Telegram Bot API
